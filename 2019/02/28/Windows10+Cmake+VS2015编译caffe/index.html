1 前期准备

  • OS: Windows10
  • CMAKE: 3.14.0-rc2-win64-x64
  • Microsoft Visual Studio Professional: 2015
  • CUDA: 8.0.61_win10
  • cuDNN: v5.1
  • Python: 3.5.2
  • boost: 1.61.0

NOTE: 在本文中Python是通过Anaconda进行配置。由于Anaconda默认环境中Python版本为3.6.5，所需要对Python进行版本降级或者新建一个虚拟环境，在本文中，采用降级Anaconda默认环境Python版本的方式，避免了在进行Cmake配置过程中需要手动添加相关Python环境路径的麻烦。
